Introduction
-------------

This project is a demonstration of the possibility of interaction with the official [VK Java SDK](https://github.com/VKCOM/vk-java-sdk)

Build JAR file
-------------
- `git clone https://github.com/C1oky/VkBot`
- `cd VkBot`
- `mvn clean package`
